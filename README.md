## Installation

for **Sublime Text 2**':

```sh
~$ git clone https://github.com/blrB/SCP-for-sintaxis-Syblime
~$ cp -a SCP-for-sintaxis-Syblime/SCP .config/sublime-text-2/Packages/SCP

```
for **Sublime Text 3**':

```sh
~$ git clone https://github.com/blrB/SCP-for-sintaxis-Syblime
~$ cp -a SCP-for-sintaxis-Syblime/SCP .config/sublime-text-3/Cache/SCP

```
